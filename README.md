## 🌟 Key Features

### 🛒 Marketplace Features
- **Course Discovery**: Browse courses by category, skill level, and ratings
- **Advanced Search**: Filter by price, duration, instructor, and popularity
- **Course Previews**: Free introductory lessons for all courses
- **Wishlist**: Save courses for later purchase

### 💳 Payment System
- **Stripe Integration**: Secure credit card payments
- **Coupon Codes**: Discount system for promotions
- **Purchase History**: Track all transactions
- **Refund Processing**: Admin-managed refund system

### 🎓 Learning Experience
- **Progress Tracking**: Course completion metrics
- **Certificate Generation**: Upon course completion
- **Q&A Section**: Per-course discussion forums
- **Bookmarking**: Save specific lessons

### ⭐ Rating System
- **5-Star Ratings**: With written reviews
- **Instructor Responses**: To student reviews
- **Verified Purchases**: Authentic review system
- **Review Moderation**: Admin controls

### 👨‍💼 Admin Dashboard
- **Course Management**: CRUD operations for all courses
- **User Analytics**: Enrollment metrics and revenue
- **Content Moderation**: Review and approve submissions
- **Revenue Reports**: Exportable financial data

## 🛠 Tech Stack

### Frontend
- **React.js** with Vite
- **Tailwind CSS** + Headless UI
- **Redux Toolkit** with RTK Query
- **React Hook Form** for forms
- **React Player** for video content

### Backend
- **Node.js** with Express
- **MongoDB** with Mongoose
- **JWT** Authentication
- **Stripe API** integration
- **Nodemailer** for emails

### DevOps
- **Docker** containerization
- **GitHub Actions** CI/CD
- **AWS EC2** deployment
- **Cloudinary** for media storage

## 🚀 Getting Started

### Prerequisites
- Node.js v16+
- MongoDB Atlas account
- Stripe developer account

### Installation

# Clone repository
git clone https://github.com/yourusername/skillkart.git
cd skillkart

# Install dependencies
npm install

# Configure environment variables
cp .env.example .env
Configuration
Update .env with your credentials:

env
MONGODB_URI=mongodb+srv://...
STRIPE_SECRET_KEY=sk_test_...
JWT_SECRET=your_jwt_secret
CLOUDINARY_URL=cloudinary://...
Running Locally
bash
# Start backend server
npm run server

# Start frontend dev server
npm run dev
